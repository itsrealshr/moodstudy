Problem Statement:
Students often choose random playlists while studying, without considering how music affects concentration, memory, or creativity.
There is no simple tool that recommends music based on mood AND study intention.

 Scope of the Project:
This project includes:
A rule-based recommendation engine
Flask backend with 3 API endpoints
A simple web frontend
SQLite storage for playlists + history
Testing through Pytest
GitHub repository with all project files

 Target Users:
Students studying for exams
Individuals who prefer mood-based productivity tools
Anyone looking for personalized study music suggestions

 Features:
Mood selection (happy, stressed, sad, neutral)
Study intention selection (focus, memorize, calm, creative)
Playlist recommendation with explanation
Pomodoro timer suggestions
Breathing exercise recommendation
History logging in SQLite
Clean UI using basic HTML
Fully open-source and easy to run
